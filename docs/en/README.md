# 📖 Alanbix Documentation (English)

Welcome to the knowledge base of **Alanbix**, the self-hosted LAN party management system powered by AI.

This documentation is organized modularly to guide you step-by-step through the installation, configuration, and usage of all the system's features.

---

## 🧭 Documentation Map

### 🚀 Quick Start
* **[Quick Start Guide](quickstart.md)**: Everything you need to deploy the system with Docker, create the first administrator account, and configure basic settings.

### 🎮 Features and Modules
* 🏆 **[Tournament Engine](features/tournaments.md)**:
  * Formats (Single, Double, Round Robin, FFA).
  * Bulk registrations, tournament teams, automatic byes (BYEs).
  * Score entry, automatic locking (🔒), and points rollback/reopening.
* 📊 **[Leaderboards and Awards](features/leaderboards.md)**:
  * Global individual leaderboard and progression indicators (↑↓ NEW).
  * Global team leaderboard (Weighted vs. Raw modes).
  * Player statistics modal and the 12 automatic end-of-LAN awards.
* 🗺️ **[Interactive Room Plan](features/room-plan.md)**:
  * Drag & Drop editor (seats, tables, logistics elements).
  * Self-service seat selection by players and cursor-centered zoom.
  * Bulk seat liberation (Bulk Unassign).
* 🤖 **[AI Assistant (Ollama)](features/ai-assistant.md)**:
  * Streaming responses (SSE) and document vectorization (RAG).
  * Priority queue (`ia_queue`) and multi-instance load balancing.
  * Message editing, inline retry, and automatic context compression.
* 💬 **[Messaging & Notifications](features/chat-notifications.md)**:
  * Quick peer-to-peer (P2P) private chats and deterministic team channels.
  * Unified notification center and match reviews generated by AI.
* 📋 **[Information Page (Markdown)](features/info-page.md)**:
  * WYSIWYG editor (EasyMDE), double view (Player vs. Projector), and detection of Windows file paths (`\\server\share`).
* 🌐 **[Language Management (i18n)](features/languages.md)**:
  * Double-layer structure (factory/user) and reactive store.
  * Bicolumn comparative editor and category filtering.
  * AI-assisted translation (Ollama) individual and in bulk.
  * One-click detection and synchronization of missing structural keys.
* 🛡️ **[System Administration & Moderation](features/administration.md)**:
  * Full profile and password management.
  * Game library and automatic cover art downloader (SearXNG).
  * AI instance monitoring and secure self-destruction buttons (Purge/Nuke).
